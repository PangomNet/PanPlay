Ohne Stream-URL kann kein Player gestartet werden.

Bitte füge der URL eines der folgenden Paramter hinzu:
  • Zum Hören einer Station von Laut.fm: ?lfmstream: NAME der Station Beispiel: Wenn die Station die URL laut.fm/eins besitzt, dann ?lfmstream=eins
  • Zum hören eines anderen Audio-Streams: ?webstream: URL
