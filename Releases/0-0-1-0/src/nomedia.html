'oOPlay' konnte nicht gestartet werden

APP-Exception: Ohne Stream-URL kann kein Player gestartet werden.

Erklärung:Sie haben 'oOPlay' ohne eine Stream-URL gestartet. Es wird jedoch eine URL zu einem HTML5-kompatiblen Audiostream benötigt. ‚oOPlay‘ bietet Ihnen zwei verschiedene Wege diese URL zu übermitteln:

Erweitern Sie die URL zum Abruf von ‚oOPlay‘ (https://oop.ownonline.eu/play.php) und ergänzen Sie eines der folgenden Parameter – selbstverständlich mit entsprechendem Wert.

  • Zum Abspielen einer Webradio-Station vom Anbieter Laut.fm: ?lfmstream= NAME der Station
    Beispiel: Die Station laut.fm/eins wird mit 'oOPlay' über folgende URL abgespielt: ?lfmstream=eins
  • Zum Abspielen eines Audiostream einer beliebigen anderen kompatiblen Abspielquelle nutzen Sie den Parameter ?webstream= URL zum Stream oder Dokument
    Beispiel: Zum Abspielen des ‚oOPlay‘-Testfiles: ?webstream=https://ownonline.eu/oop/src/assets/audio/winxp11.mp3
    Oder zum Abspielen eines andere Audio-Livestreams: ?webstream=https://icecast.omroep.nl/radio2-bb-mp3
